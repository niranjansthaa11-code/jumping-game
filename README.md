# 🧗‍♂️ Don't Look Down

A fast-paced vertical platformer built with **Godot 4.x**. </br>
Test your reflexes </br>
THe goal is to simply : climb as high as you can, and try not to fall into the void.



## 🎮 Features
* **Infinite Verticality:** How high can you climb before the void catches you?
* **Fearful Platforms:** Includes "Fake Floors" that collapse shortly after being touched Which are unknown .

## 🛠️ Technical Highlights
This project demonstrates several core Game Development concepts:
* **Singletons (Autoloads):** Uses a `ScoreManager` node to track data across scenes (Game → Death Menu → Restart).
* **Collision Logic:** Advanced use of `move_and_slide()` and `get_slide_collision()` to handle platform interaction and scoring.



## ⌨️ Controls
| Action | Input |
| :--- | :--- |
| **Move Left** | `A` or `Left Arrow` |
| **Move Right** | `D` or `Right Arrow` |
| **Jump** | `Space` or `W`or `Up Arrow` |
